# Django Blog

Это проект блога, разработанный с использованием Django. Он включает функциональность для создания, редактирования и удаления постов, а также добавления комментариев. Проект создан в рамках обучения Django и является хорошей практикой для освоения основ фреймворка.

## Описание

Проект включает следующие функции:

- Просмотр списка постов
- Фильтрация постов по категориям и местоположениям
- Регистрация пользователей и управление профилем
- Создание, редактирование и удаление постов
- Добавление, редактирование и удаление комментариев к постам
- Админ-панель для управления категориями, местоположениями и постами
